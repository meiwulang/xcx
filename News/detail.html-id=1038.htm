廷邦小程序社交裂变工具全新上线！

来源：微圈互动   时间：2018-06-12

小程序自面世以来，越来越多营销工具的引入，使其引流裂变能力进一步提升。根据试点数据显示，小程序通过分享裂变率达150%，获客成本低至1.7元，发券核销率超过20%。

小程序社交立减金

廷邦小程序响应商家需求，正式上线小程序社交立减金功能，通过用户门店消费赠送社交立减金，利用微信好友间的社交裂变，达到低成本拓客、高效转化的目的。

如何获得社交立减金?

用户A在线下某生鲜门店进行消费后，支付时采用微信支付扫码付款(包括扫描付款二维码以及商家扫码)，付款成功后，会收到一定数量的社交立减金，邀请好友一起领取，即可获得一份社交立减金，同时该生鲜门店也得到了更大机会的曝光。

当再次在该生鲜门店使用微信支付消费时，微信会直接抵扣该社交立减金。而领取到立减金的用户及其好友，会积极进行下一次消费，提高了商品转化率和消费积极性。

该营销功能对于销售低价、高频商品的商家效果更好，比如餐饮、外卖等日常消费。

社交立减金的价值

1、分享裂变

社交立减金功能，充分契合了微信的社交属性，通过分享得立减金调动了分享者以及领取者的积极性。基于这种分享机制，商家每获得一个用户，就可以通过该用户裂变一定数量的新用户，带来的是裂变级的推广效果，有效增加小程序曝光量。

2、与卡包相连

用户领取的社交立减金可在微信卡包内查看，将小程序与微信卡包功能进一步打通，让用户在微信里的“消费-分享-消费”的流程更顺畅。

3、降低拉新成本

小程序内完成支付后，商家向用户赠送购物“立减金”，用户必须分享给好友才能领取，好友领取之后可直接进行新一轮消费。借助微信的社交能力，通过好友分享提高转化率，降低拉新成本。

社交立减金设置

廷邦小程序社交立减金的设置分为两步：

1. 创建社交立减金卡券

组件 → 微信卡劵 → 创建“微信社交立减金卡劵”

创建卡券时，商家需设置好卡券的使用条件——消费门槛，这将决定了社交立减金的发放门槛，当用户消费金额达到商家设置的金额时，会发放社交立减金。同时，设置卡券的有效使用期限，用户需在有效期内进行消费。

微信卡劵创建成功后，需登录“商户后台”激活卡劵。

激活路径：营销中心 → 管理代金劵 → 草稿箱 → 详情 → 确认激活。

2. 创建“微信社交立减金”

组件 → 微信社交立减金。

1)操作微信支付免充值模式验收

2)创建社交立减金

进行相应设置后，需登录“商户后台”激活微信社交立减金。

激活路径：营销中心 → 营销活动 → 满额送→ 管理→ 激活。

获客新玩法：摇一摇领红包

本次上线的新功能中，除了社交立减金是商家的吸客利器，摇一摇领红包活动也是商家提高产品转化率的重要工具。

商家可以在小程序首页设置摇一摇领红包活动，用户打开小程序即可参与该活动，红包的形式可以是微信零钱、积分以及储蓄卡余额，商家可以根据实际情况选择红包形式以及额度，有效提高用户参与度。

该活动也可以分享给好友，好友参与后也可获得相应奖励，从而提高小程序的引流和拓客能力。

活动设置：组件 → 营销 → 摇红包

添加摇红包活动后，通过后台的店铺装修链接到该活动，用户即可在小程序页面查看该活动信息。

小程序(平台版)新增功能(5项)

规则：

①邀请才可成为团友，邀请人为团长，满足人数才可开团。为了避免团长开团后长期不能成团，开团24小时后如未成团，将会自动关闭订单并退款。如果设置项“是否自动成团”选择“是”，则24小时后自动成团，无论人数达到团购人数。

②会员目前只能选择一个数量的商品发起拼团，并不能参自己的团。

③会员如果参团成功，则不能二次参团。

操作流程：

门店后台--组件--拼团--拼小团，添加小团商品。

参团流程：

规则：

1、两种送券方式，即下单支付即送和订单完成后赠送。两种方式支持配送及自提订单，因货到付款订单下单时不涉及到支付，所以仅支持订单完成后赠送券。

2、目前使用场景仅支持“商品购买”场景，只要满足规则中任一条件即送券。

3、单店下单满足规则即送，多店下单只要满足一个店铺规则即送。

4、活动规则目前仅支持添加6条数据。

操作流程：

门店后台--组件--营销模块--消费送券，开启活动按钮及设置营销规则。

操作流程：

商圈后台--门店--门店列表--选择某一门店，绑定用户(一个门店只能绑定一个会员信息) 选择会员绑定成功，小程序可直接查看门店信息。

门店后台—门店设置—选择门店展示样式

复杂版or简单版

复杂版：底部导航可根据后台设置来展示，并且可以展示推荐单品，门店活动，优惠券，套餐优惠，用户评价等信息。

简单版：底部导航固定为(主页，动态，介绍，我的)并且只展示门店的简介,营业时间,门店图标,同城文章动态等简单信息。
